Штаб КР по COVID-19

Приложение для отслеживания ситуации с COVID-19 в Кыргызстане



Часто задаваемые вопросы:

Почему бы не загрузить приложение на плеймаркет?

Политика конфиденциальности плеймаркета не позволяет нам загрузить приложения связанные с войной и трагическими событиями в плеймаркет

Почему выходит предупреждение что приложение может быть опасно для моего смартфона?

Это опять же связано с тем, что политика Android подразумевает что сторонние приложения могут быть опасны для смартфона без надлежащей проверки. Из-за этого могут появляться предупреждения.
