Политика конфиденциальности обработки данных для Приложения QR Code Tunduk

1. Общие положения

1.1 Использование мобильного приложения QR Code Tunduk означает согласие с настоящей Политикой конфиденциальности и условиями обработки персональных данных Пользователя

1.2 2.2. В случае несогласия с условиями Политики конфиденциальности Пользователь должен прекратить использование мобильного приложения.

2. Предмет политики конфиденциальности

2.1 Приложение не обрабатывает и не хранит никаких персональных данных собранных с помощью камеры или полей для ввода текста

2.2 Приложение использует камеру для обработки и нахождения QR кодов на изображении после обнаружения QR кода все фотографии удаляются

2.3 Приложение использует камеру только при необходимости и не хранит никаких данных с камеры и не отправляет их на сервер

2.3 В приложении хранятся последние 20 отсканированных кодов. Данные не покидают приложение
